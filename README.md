## 概要：Overview
飲食店予約サービス
主な機能
・会員登録
・ログイン
・ログアウト
・ユーザー情報取得
・ユーザー飲食店お気に入り一覧取得
・ユーザー飲食店予約情報取得
・飲食店一覧取得
・飲食店詳細取得
・飲食店お気に入り追加
・飲食店お気に入り削除
・飲食店予約情報追加
・飲食店予約情報削除
・飲食店検索（エリア/ジャンル/店名）

・飲食店評価機能
・予約情報変更機能実装
・管理者権限（新規飲食店代表者登録）
・店舗代表者権限（新規飲食店登録、自店舗予約確認）
